# 장단기 금리 역전이란?

보통은 돈을 오래 빌려줄수록(10년) 이자를 더 많이 받지만, 경기가 안 좋아질 것 같으면 단기(2년) 금리가 더 높아지는 기현상이 발생합니다. 이를 "금리 역전"이라 하며, 역사적으로 경기 침체(Recession)의 강력한 전조로 해석됩니다.

# 왜 인플레이션과 함께 보나요?

물가(인플레이션)가 오르면 중앙은행은 이를 잡기 위해 기준 금리를 올립니다. 이때 단기 금리는 정책에 민감하게 반응하여 급등하지만, 장기 금리는 "앞으로 경기가 둔화될 것"이란 우려로 덜 오르거나 떨어집니다. 즉, 인플레이션 → 고강도 긴축 → 금리 역전 → 경기 침체로 이어지는 연결 고리를 확인하기 위함입니다.

# 이 그래프로 무엇을 알 수 있나요?

붉은색 영역(역전 구간): "경고등"이 켜진 시기입니다. 보수적인 투자 관점이 필요합니다.

인플레이션(빨간 선) 추세: 물가가 잡히고 있는지, 아니면 여전히 높아 고금리가 유지될지 가늠할 수 있습니다.
